moodle-local_intelliboard
==================

A local Moodle Module plugin

- Track and analyze your Moodle data in real time


INSTALLATION

1. Download the plugin and unzip it in moodle/local/intelliboard/

2. Install the plugin

3. Enable web service advance feature (Admin > Advanced features)

4. Enable REST protocol (Admin > Plugins > Web services > Manage protocols)

5. Add new Authorized users (Admin > Plugins > Web services > Manage services > Authorized users > click on user full name)

6. Create a token for a ADMIN user and for the service 'IntelliBoard service' (Admin > Plugins > Web services > Manage tokens)
